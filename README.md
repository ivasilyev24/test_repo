*    gradle build (из idea)
*    проверялось на java 11
*    Для запуска приложения выполните:
    java -Dfile.encoding=UTF-8 -jar <дирректоия проекта>\build\libs\test7-1.0-SNAPSHOT-all.jar

    <дирректоия проекта>\src\main\resources\out_litle.json
    <дирректоия проекта>\src\main\resources\out_litle.csv
